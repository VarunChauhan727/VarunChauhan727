Exercise 5: Exploratory Data Analysis and Insight Generation

Overview
This project applies exploratory data analysis (EDA), correlation analysis, and customer segmentation
to identify key drivers of campaign response and generate actionable business insights.

Key Techniques
- Correlation analysis on target variable
- Exploratory Data Analysis (EDA)
- K-Means clustering and segmentation
- Data visualization using Matplotlib and Seaborn

Tools Used
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn

Output
- Identification of variables associated with customer response
- Clear customer segments with distinct response behavior
- Business-ready insights to support data-driven decision making
